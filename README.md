# E-Commerce Credit Risk Dashboard

## Overview

An interactive credit risk assessment tool built using Python and Streamlit for SME and e-commerce lending analysis.

The application evaluates borrower financial metrics and generates a credit score, risk rating and funding recommendation using a rules-based underwriting framework.

## Features

* Credit scoring framework
* Risk rating classification
* Funding recommendation engine
* Financial ratio analysis
* Scenario stress testing
* Automated credit commentary

## Technologies Used

* Python
* Streamlit
* Pandas
